# Script starter

A lightweight scaffolding tool to instantly generate new Node.js script - batteries-included.

## Prerequisites
Ensure you have the following available in your terminal:
- Node.js & npm
- IntelliJ IDEA (with the idea command-line launcher configured)
- Bash environment

## Initial Setup
1. Set Permissions (Required only once): `chmod +x generate-node-script.sh`
2. Update bash script with destination folder in which new projects will be created: `../generate-node-script.sh:19`

## Usage
Follow these steps to generate a new project:
1. Execute the Generator: `./generate-node-script.sh`
2. Enter your project name when prompted by the CLI.

What happens next:
- A new folder with the specified project name will be created in the configured destination
- The template script will be copied into this folder
- Dependencies will be installed automatically
- The project will be opened in IntelliJ IDEA for immediate development
